Features

あなたの開発を、加速する

iPhoneのどこからでも、Claude Codeのフル機能を手のひらで使えます。

⚡

リアルタイムコード生成

自然言語で指示するだけで、すぐに動くコードを生成。TypeScript・Python・Swift など主要言語に対応。

🔍

スマートコードレビュー

コードを貼り付けるだけでバグ・セキュリティリスク・パフォーマンス問題を即座に発見し、改善案を提示。

🐛

インテリジェントデバッグ

エラーメッセージやスタックトレースを共有すると、原因を特定して修正方法をステップごとに説明。

📖

コード解説・ドキュメント生成

複雑なコードを即座に日本語で解説。JSDoc・コメント・READMEの自動生成にも対応。

🔗

GitHub 連携

リポジトリやPRと直接連携。コードレビューのコメントをその場でAIに相談できます。

🔒

エンタープライズセキュリティ

コードはデバイスとAnthropicの間でエンドツーエンド暗号化。SOC 2 Type II 準拠。

コード出力

ハイライト・コピー・共有

①

入力：自然言語 → 構造化プロンプト

ユーザーのメッセージ・貼り付けコード・会話履歴を組み合わせてコンテキストウィンドウを構築。言語・フレームワークを自動判定します。

②

推論：大規模言語モデルによるコード理解

Anthropic の Claude（Sonnet / Opus）が数百億パラメータで構文・意図・ベストプラクティスを同時に考慮し回答を生成します。

③

出力：ストリーミングで即時表示

回答はトークン単位でストリーミング配信。長いコードも待ち時間なく先頭から読み始められます。

④

セキュリティ：コードはデバイス外に保存しない

通信はTLS 1.3で暗号化。会話ログはセッション終了後に削除され、第三者のモデル学習には使用されません。

How It Works

3ステップで始められる

インストールから最初のコード生成まで、1分もかかりません。

1

アプリをダウンロード

App Storeから無料でインストール。既存のAnthropicアカウントでサインインするだけ。

2

Claudeに話しかける

「このコードをリファクタリングして」「バグを直して」など、日本語で自由に指示できます。

3

コードをコピー＆活用

生成されたコードをワンタップでコピー。IDEや GitHub に即座に貼り付けて使えます。
